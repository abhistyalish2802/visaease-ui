# VisaEase - Student Visa Application UI

VisaEase is a user-centered web interface designed to simplify the student visa application process.

## Features

- Step-by-step visa application process
- Personal information form
- Study details section
- Document upload system
- Review and submission page
- Confirmation screen

## UX Principles Applied

- Minimalist interface
- Clear navigation
- Progress indicators
- Error prevention
- User reassurance

## Technologies Used

- HTML
- CSS
- JavaScript
- Visual Studio Code
- GitHub
